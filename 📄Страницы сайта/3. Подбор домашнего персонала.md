## 3.1 ПЕРВЫЙ БЛОК

**Хлебные крошки, навигация:** Главная → Услуги → Домашний персонал
**Заголовок:** Домашний персонал для вашей семьи
**Подзаголовок:** Тщательный отбор. Проверенные кандидаты. Конфиденциальность на каждом этапе. Без комиссии для персонала.
**Кнопка:** «Записаться на консультацию» → ведёт к форме заявки

## 3.2 ВЫБЕРИТЕ ПОЗИЦИЮ ДЛЯ ПОДБОРА

**Заголовок:** С кем помочь?
**Подзаголовок:** Нажмите на нужную позицию, чтобы узнать подробнее об условиях подбора.

**Сетка карточек с фото (3 колонки):** _(Каждая карточка: фото + название + кнопка «Подробнее»)_
Няня к младенцу / Няня-воспитатель / Гувернантка / Репетитор / Домработница / Няня за границу / Повар / Тьютор / Стилист / Логопед / Семейный водитель / Личный ассистент / Бизнес-ассистент / Администратор / Догситтер / Управляющий домом / Горничная / Личный водитель / Садовник / Охранник / Личный тренер / Массажист / Косметолог / Парикмахер / Специалист по ремонту / Организатор мероприятий / Флорист

> **Как открываются карточки?** 
> При клике открывается модальное окно с:
> - названием позиции
> - кратким описанием
> - обязанностями
> - условиями подбора

- Как к вам обращаться? _(поле: Ваше имя)_
