## 3.1 ПЕРВЫЙ БЛОК

**Хлебные крошки, навигация:** Главная -> Услуги -> Домашний персонал
**Заголовок:** Подбор домашнего персонала для вашей семьи

**Подзаголовок:** С теплом и заботой подберём идеальных специалистов для вашей семьи. Тщательно проверяем каждого кандидата. Без комиссии для персонала.
**Кнопка:** «Оставить заявку на подбор» - ведет к форме заявки

## 3.2 ВЫБЕРИТЕ ВАКАНСИЮ ДЛЯ ПОДБОРА

**Заголовок:** Выберите вакансию для подбора
**Подзаголовок:** Нажмите на нужную вакансию, чтобы узнать подробнее об условиях подбора.

**Сетка карточек с фото (3 колонки):**
*(Каждая карточка: фото + название + кнопка «Посмотреть подробнее»)*

Няня к младенцу / Няня-воспитатель / Гувернантка / Репетитор / Домработница / Няня за границу / Повар / Тьютор / Стилист / Логопед / Семейный водитель / Личный ассистент / Бизнес-ассистент / Администратор / Догситтер / Управляющий домом / Горничная / Личный водитель / Садовник / Охранник / Личный тренер / Массажист / Косметолог / Парикмахер / Специалист по ремонту / Организатор мероприятий / Флорист

> **Как вакансии будут открытьвася?** Вакансии будут оформлены так же в виде карточек с названием и кнопкой подробнее. 
> При клике на кнопку будет открываться окно с доп.информацией: 
> - название вакансии,
> - небольшое описание ,
> - обязанности,
> - условия работы

## 3.3 ФОРМА ЗАЯВКИ НА ПОДБОР ПЕРСОНАЛА

**Заголовок:** Оставьте заявку на подбор персонала
**Подзаголовок:** Мы расскажем, как происходит процесс подбора, и сориентируем вас по цене. Без обязательств.

**Поля формы:**
- Как к вам обращаться? *(поле: Ваше имя)*
- Ваш номер телефона *(поле: +7)*
- Какие вакансии для подбора вас интересуют? *(выпадающий список или поле)*
- Введите ваше сообщение... *(текстовое поле)*
**Кнопка:** «Оставить заявку на подбор» 
**Мелкий текст под кнопкой:**
> Нажимая на кнопку, вы даёте согласие на обработку персональных данных и соглашаетесь с политикой конфиденциальности.
