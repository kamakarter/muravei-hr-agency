## 4.1 ПЕРВЫЙ БЛОК

**Хлебные крошки, навигация:** Главная -> Услуги -> Персонал для бизнеса
**Заголовок:** Подбор персонала для вашего бизнеса
**Подзаголовок:** Подберём надёжный персонал для вашего бизнеса с гарантией замены и без комиссии для кандидатов.
**Кнопка:** «Оставить заявку на подбор» - ведет к форме заявки

## 4.2 ВЫБЕРИТЕ ВАКАНСИЮ ДЛЯ ПОДБОРА

**Заголовок:** Выберите вакансию для подбора

**Сетка карточек (3 колонки):**
Менеджер по продажам / Руководитель отдела продаж / Бухгалтер / Финансовый аналитик / Оператор колл-центра / Офис-менеджер / Главный прораб / Экономист / Специалист отдела кадров / Младший линейный персонал / Финансовый директор / Графический дизайнер / Организатор мероприятий / Менеджер по закупкам / Повар / Кондитер

> **Как вакансии будут открытьвася?** Вакансии будут оформлены так же в виде карточек с названием и кнопкой подробнее. 
> При клике на кнопку будет открываться окно с доп.информацией: 
> - название вакансии,
> - небольшое описание ,
> - обязанности,
> - условия работы

## 3.3 ФОРМА ЗАЯВКИ НА ПОДБОР ПЕРСОНАЛА

**Заголовок:** Оставьте заявку на подбор персонала
**Подзаголовок:** Мы расскажем, как происходит процесс подбора, и сориентируем вас по цене. Без обязательств.

**Поля формы:**
- Как к вам обращаться? *(поле: Ваше имя)*
- Ваш номер телефона *(поле: +7)*
- Какие вакансии для подбора вас интересуют? *(выпадающий список или поле)*
- Введите ваше сообщение... *(текстовое поле)*
**Кнопка:** «Оставить заявку на подбор» 
**Мелкий текст под кнопкой:**
> Нажимая на кнопку, вы даёте согласие на обработку персональных данных и соглашаетесь с политикой конфиденциальности.
