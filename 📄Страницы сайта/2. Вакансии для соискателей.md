## 2.1 ПЕРВЫЙ БЛОК

**Хлебные крошки, навигация:** Главная -> Вакансии
**Заголовок:** Наши вакансии для домашнего персонала
**Подзаголовок:** Предоставляем вакансии для домашнего персонала без комиссии и с полной поддержкой трудоустройства.
**Кнопка:** «Оставить заявку на работу»

---

## 2.2 КОГО МЫ ИЩЕМ (карточки с вакансиями)

**Заголовок:** Кого мы ищем?
**Подзаголовок:** Выберите подходящую вакансию и оставьте заявку — мы свяжемся с вами в ближайшее время.
**Сетка карточек с фото и названием (4 колонки):**
Няня к младенцу / Няня-воспитатель / Гувернантка / Репетитор / Домработница / Повар / Тьютор / Стилист / Логопед / Семейный водитель / Личный ассистент / Бизнес-ассистент / Администратор / Догситтер / Управляющий домом / Горничная / Личный водитель / Няня за границу / Садовник / Охранник / Личный тренер / Массажист / Косметолог / Парикмахер / Специалист по ремонту / Организатор мероприятий / Флорист


> **Как вакансии будут открытьвася?** Вакансии будут оформлены так же в виде карточек с названием и кнопкой подробнее. 
> При клике на кнопку будет открываться окно с доп.информацией: 
> - название вакансии,
> - небольшое описание ,
> - обязанности,
> - условия работы

## 2.3 ФОРМА ЗАЯВКИ «ХОЧУ РАБОТАТЬ» 

**Заголовок:** Заполните заявку для отклика на вакансию
**Подзаголовок:** Заполните форму и мы свяжемся с вами в ближайшее время. Помогаем с трудоустройством без комиссии.
**Поля формы:**
- Как к вам обращаться? *(поле: Ваше имя)*
- Ваш номер телефона *(поле: +7)*
- Укажите желаемую должность *(поле ввода или список)*
- Расскажите о вашем опыте работы *(поле: введите информацию о себе или вставьте ссылку на резюме)*
**Кнопка:** «Оставить заявку на работу» 
**Мелкий текст:**
> Нажимая на кнопку, вы даёте согласие на обработку персональных данных и соглашаетесь с политикой конфиденциальности.
